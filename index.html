SafeDrop

SafeDrop is a minimal, self-hosted file-sharing platform that lets users upload files and generate shareable links with built-in expiration logic — no logins, no storage bloat, no unnecessary complexity. Built without Docker and deployed on Vercel for seamless performance, SafeDrop is designed to offer a smooth user experience. Users can simply upload a file and get a unique link valid for a specific number of hours or days. Unlike Google Drive links (which stay active unless manually revoked), SafeDrop offers one simple but powerful improvement: Automatic Link Expiry — Once the set time limit passes, no one (even with the link) can access the file anymore.
